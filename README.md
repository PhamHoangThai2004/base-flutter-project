# base-flutter-project
This is an example mobile project used to base flutter

# Instruct:
1) Execute command: `$ flutter doctor`
2) Execute command: `$ flutter pub get`
3) Execute command: `$ flutter pub run build_runner build --delete-conflicting-outputs`
4) Execute command: `$ flutter run` to install app

# Gen code:
- Gen assets: Execute command: `$ dart run build_runner build`
